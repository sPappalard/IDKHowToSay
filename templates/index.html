📋 File Format Requirements

📁 Supported File Types

  • Excel files: .xlsx, .xls formats
  • CSV files: .csv format with UTF-8 encoding
  • Google Sheets: Public or shared via link
🏗️ Required Structure

Your file must have exactly 2 columns with the language pairs you want to practice.

📊 File Structure

Column 1 (First Language) Column 2 (Second Language)
house casa
car macchina
book libro
water acqua
🔄 Multiple Translations

You can provide alternative translations separated by: / , \ ( )

✨ Advanced Examples

English Italian
house/home casa
car, automobile macchina, auto
beautiful bello/bella (belli/belle)
to eat mangiare
🔗 Google Sheets Setup

1. Create your sheet with the format above
2. Click "Share" → "Anyone with the link can view"
3. Copy the link and paste it in the input field
4. The system will automatically extract your data

⚠️ Important Notes

• Make sure language selections match your file columns
• First column = First Language, Second column = Second Language
• Empty rows will be automatically skipped
